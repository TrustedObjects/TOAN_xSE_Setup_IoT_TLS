=======================
TO-Protect (evaluation)
=======================

In this folder, you can find TO-Protect TLS evaluation binary to be flashed on
your product.
This version of TO-Protect is compatible with Cortex-M family except Cortex-M0/M0+/M1/M23.
If you need other architecture (including Cortex-M0/M0+/M1/M23), please contact Trusted Objects:
	https://www.trusted-objects.com/contact-us.html


You can also find some evaluation SecureStorage to personalize your TO-Protect.

Note that TO-Protect evaluation version do not embed security hardening.
